\begin{abstract}
Text-based machine learning systems are usually trained on past data but deployed on new data, whose patterns may shift over time through what is called concept drift or semantic drift. The problem is harder when no labels are available for performance monitoring. This work presents an unsupervised framework for detecting semantic drift in time-stamped text using embedding-based representations. The pipeline maps text into dense vectors, groups them into chronological windows, and tracks how the embedding distribution changes over time. The experiments use $308{,}313$ time-stamped IMDB reviews and compare three sentence encoders: all-MiniLM-L6-v2, LaBSE, and a multilingual DistilBERT. The framework combines autoencoder reconstruction error with dimension-wise Kolmogorov--Smirnov and Anderson--Darling tests, Jensen--Shannon divergence, and Sliced Wasserstein Distance in both the original embedding space and the autoencoder latent space. The main finding is that the autoencoder reconstruction-error signal is consistent across encoders, with year-by-year Pearson correlations of $r \geq 0.97$ on post-gap aggregates. The dimension-wise and bootstrap-based distance tests flag drift in every post-gap year at this corpus scale, which limits their value as discriminators between years.
\end{abstract}

\section{Introduction}
\label{sec:introduction}

Text data is central to many modern machine learning applications, including sentiment analysis, topic modeling, information retrieval, content moderation, recommendation, and conversational agents. These systems are usually trained on past data and deployed under the assumption that future input will follow similar linguistic and semantic patterns. In real-world environments, this assumption often fails: new topics appear, terminology changes, user behavior evolves, and external events influence how people write.

When the distribution of incoming data changes over time, the performance of a machine learning system may drop. This effect is commonly called concept drift \cite{gama2014,lu2018}; in text systems it appears as semantic drift, where the meaning or contextual distribution of words and documents shifts over time. A common difficulty in detecting drift is that many real-world text pipelines do not have continuous access to labeled data, so performance-based monitoring is not usable. In unlabeled settings, drift detection must rely on changes in the input distribution itself \cite{abghani2020,demello2019}.

Most existing work in the streaming and tabular setting focuses on labeled or low-dimensional inputs, where simple distance tests on raw features perform well. Modern text pipelines, in contrast, work on high-dimensional dense embeddings whose statistical behavior is less well understood \cite{rabanser2019}, and where the same nominal test can behave very differently across corpus scales. This gap between standard drift-detection theory and practical text monitoring at scale motivates a careful comparison of several unsupervised drift signals on the same corpus and across several embedding models.

This project studies semantic drift through embedding-based text representations. Each document is mapped into a dense vector that carries semantic information \cite{mikolov2013,le2014,reimers2019}, and chronological time windows are compared in this vector space. The main contributions are:

\begin{itemize}
    \item A reproducible and scaled-up pipeline for detecting semantic drift in unlabeled time-stamped text data, evaluated on $308{,}313$ IMDB reviews.
    \item A comparison of three sentence-embedding representations (all-MiniLM-L6-v2, LaBSE, multilingual DistilBERT) that shows the autoencoder reconstruction-error signal is consistent across them.
    \item An evaluation of dimension-wise (KS, Anderson--Darling), distance-based (Sliced Wasserstein), and divergence-based (Jensen--Shannon) methods at scale, including the behavior of each signal individually.
    \item A discussion of the sensitivity, stability, and interpretability trade-offs in unsupervised text drift monitoring, including a concrete analysis of why some standard tests degrade at this corpus scale.
\end{itemize}

The remainder of the paper is organized as follows. Section~\ref{sec:problem} states the problem. Section~\ref{sec:background} reviews related work. Section~\ref{sec:methodology} describes the framework and experimental setup. Section~\ref{sec:results} presents the results together with the analysis and validation against the project objectives. Section~\ref{sec:discussion} discusses the findings, the limitations, and the code and data availability of the project. Section~\ref{sec:conclusion} concludes.

\section{Problem Statement}
\label{sec:problem}

Machine learning models for text are usually trained on a fixed historical dataset and applied to new incoming data. This implicitly assumes that the training distribution $P_{\text{train}}(X)$ and a later distribution $P_t(X)$ at time $t$ are similar:
\begin{equation}
P_{\text{train}}(X) \approx P_t(X).\label{eq:assumption}
\end{equation}

In real-world text environments this often fails, and the distribution of incoming text moves away from the historical training data:
\begin{equation}
P_{\text{train}}(X) \neq P_t(X).\label{eq:drift}
\end{equation}

This can reduce the reliability of downstream models. In supervised settings, the drop can be observed through metrics such as accuracy or F1-score, but in many practical text systems, labels for new data are not available, so performance-based monitoring is not possible \cite{abghani2020}.

The main problem addressed in this project is the detection of semantic drift in unlabeled, time-stamped text data. Each document is turned into a dense embedding, and the embeddings are grouped into chronological windows. The task is to decide whether the embedding distribution of a later window differs noticeably from the historical reference distribution, without relying on ground-truth drift labels. Several practical issues arise: text embeddings are high-dimensional, making distribution comparison sensitive to parameter choices; semantic drift may be gradual rather than sudden, so small changes accumulate over time \cite{gama2014,lu2018}; and detected signals cannot be checked against downstream accuracy.

The project is guided by three research questions: how can semantic drift be detected over time without labels; how do embedding-based representations reflect semantic change, and how stable are these signals across encoders; and which unsupervised drift detection techniques are most useful for text embeddings at scale.

\section{Background and Related Work}
\label{sec:background}

\subsection{Concept Drift and Semantic Drift in Text}

Concept drift is a change in the statistical properties of data over time \cite{gama2014,lu2018}. It is a central problem in machine learning systems that operate in dynamic environments, because models trained on past data may no longer fit later data. When labels are available, the change can be tracked through performance metrics; otherwise, drift detection must rely on changes in the input distribution itself. Concept drift may be sudden, gradual, incremental, or recurring \cite{gama2014}, and text data is especially likely to show gradual or recurring change as language evolves continuously. In text-based systems, drift often appears as semantic drift: the meaning, usage, or contextual distribution of words, phrases, or documents changes over time. A related line of work in computational linguistics studies diachronic semantic change directly by aligning word embeddings trained on different time slices and analyzing how individual words move in the resulting space \cite{hamilton2016,kutuzov2018}. Previous work has also explored drift detection in text streams using document embeddings, which supports the idea that semantic changes can be tracked through distributional changes in embedding spaces \cite{hidalgo2019}.

\subsection{Embedding-Based Text Representations}

Embedding models turn text into dense numerical vectors so that documents with similar meanings lie close together in the vector space. Compared with sparse representations such as bag-of-words or TF-IDF, dense vectors encode semantic similarity in continuous spaces \cite{mikolov2013,le2014}. Earlier approaches such as Word2Vec introduced efficient neural methods for learning continuous word representations from context \cite{mikolov2013}, and Doc2Vec extended this idea to sentences, paragraphs, and documents \cite{le2014}. More recent transformer-based architectures \cite{vaswani2017} and the BERT family of pre-trained encoders \cite{devlin2019} produce contextual representations that have become the standard backbone for sentence embeddings \cite{reimers2019}. This project uses three such encoders, which differ in size, in whether they are distilled from a larger model \cite{sanh2019}, and in whether they are trained for monolingual or multilingual sentence representations \cite{feng2020}. The encoder family was chosen to span a small monolingual distilled model, a large multilingual sentence model, and a multilingual distilled BERT, so that the comparison reflects different points on the size and multilinguality axes rather than three close variants of the same architecture.

\subsection{Unsupervised Drift Detection}

Unsupervised drift detection identifies changes in the input distribution without labels and is important in real-world systems where labels are not available immediately \cite{abghani2020,demello2019}. Statistical distance measures are commonly used: distribution-based methods compare past and new data using a dissimilarity measure or a statistical test \cite{lu2018}. Wasserstein distance measures how much one distribution must be transported to match another, and a sliced version computed over random one-dimensional projections gives a scalable approximation in high-dimensional settings \cite{bonneel2015}. Divergence-based measures such as Jensen--Shannon divergence can also be used. The reliability of these methods depends on choices such as window size, embedding model, dimensionality reduction, and threshold selection. Recent surveys note that the practical performance of these tests is strongly sample-size dependent: when the reference and current windows are both large, even small but systematic shifts produce overwhelming evidence against the null \cite{lu2018,bayram2022}. A complementary line of work uses multivariate alternatives such as the Maximum Mean Discrepancy (MMD) kernel two-sample test \cite{gretton2012}, which avoids the dimension-wise inflation of univariate tests at the cost of additional kernel-choice and bandwidth assumptions.

\subsection{Autoencoder-Based Drift Detection}

Another approach uses the reconstruction error of an autoencoder trained on a reference period, a strategy closely related to autoencoder- and variational-autoencoder-based anomaly detection \cite{ancho2015}. The autoencoder maps each embedding into a lower-dimensional latent representation and then reconstructs it; the reconstruction error is
\begin{equation}
e_i = \|z_i - \hat{z}_i\|^2,\label{eq:recon}
\end{equation}
where $z_i$ is the input embedding and $\hat{z}_i$ is the autoencoder reconstruction. If the autoencoder is trained only on the reference period, it should reconstruct similar future embeddings with relatively low error and produce higher error when later embeddings come from a shifted semantic distribution. Changes in the average reconstruction error over time therefore serve as an unsupervised drift signal. Compared with raw distributional tests, this approach has two practical advantages: it produces a per-sample, real-valued score that aggregates smoothly into a yearly mean, and it does not rely on a hypothesis-testing framework that may become trivially significant at scale.

\section{Methodology}
\label{sec:methodology}

\subsection{Overall Pipeline}

The framework follows an unsupervised pipeline with five steps: (i) preprocessing the text data and parsing timestamps; (ii) computing dense embeddings under three different sentence encoders; (iii) splitting the data into chronological windows, with the earliest years used as a reference; (iv) training an autoencoder on the reference period for each encoder; and (v) comparing each later window to the reference using both autoencoder reconstruction error and statistical distance-based measures.

Each year is treated as a separate window, and the embeddings of all reviews from that year are read as a sample from the semantic distribution of that period. The reference distribution is fixed once and used as the comparison anchor for all later years. For every later year, the autoencoder reconstruction error gives a graded per-sample signal that aggregates into a yearly mean, while the distance-based measures (Jensen--Shannon, Kolmogorov--Smirnov, Anderson--Darling, and Sliced Wasserstein Distance) compare the yearly embedding distribution directly to the reference. The same Sliced Wasserstein procedure is also applied to the autoencoder's latent representation, which gives a slightly denoised view of the same comparison.

\subsection{Justification of Method Choices}

The methods were chosen to cover four complementary families of unsupervised drift signal that are commonly cited in the literature \cite{lu2018,abghani2020,bayram2022}, so that their behavior can be compared on the same corpus:

\begin{itemize}
\item \textbf{Autoencoder reconstruction error} as a learned, per-sample anomaly score that does not depend on a hypothesis test. This was preferred over simpler probabilistic models (e.g.\ Gaussian fits on PCA components) because the data are high-dimensional and not Gaussian, and over downstream classifier confidence because no labels are available.
\item \textbf{Kolmogorov--Smirnov and Anderson--Darling} tests as well-known non-parametric two-sample tests \cite{abghani2020}. They are simple and dimension-wise, and they give a clear yes/no answer per dimension that can be aggregated through false-discovery-rate control.
\item \textbf{Sliced Wasserstein Distance} as a multivariate distance that is feasible in high dimensions \cite{bonneel2015}. SWD avoids the curse of dimensionality that affects classic Wasserstein distance, and unlike KS or AD it produces a single multivariate distance value rather than a per-dimension test.
\item \textbf{Jensen--Shannon divergence} as a bounded, symmetric divergence often used in drift work. It is included to compare a divergence-based signal with the distance-based and learned alternatives.
\end{itemize}

Each drift signal is interpreted along three dimensions: \emph{sensitivity}, whether changes are detected early; \emph{stability}, whether the signal avoids reacting to small short-term fluctuations; and \emph{interpretability}, whether the signal can be understood in terms of the underlying data. Since the pipeline is unsupervised, the detected signals are read as indicators of distributional change rather than as proof of downstream model failure.

\subsection{Dataset and Temporal Split}
\label{sec:experimental}

The experiments are run on a time-stamped IMDB review dataset, building on the IMDB movie-review corpus introduced by Maas et al.\ \cite{maas2011} and extended in this work with parsed review timestamps. After preprocessing and removing records with invalid dates, the dataset contains $308{,}313$ encoded reviews.     \item A \textbf{test period} (years after 2010) used for post-gap drift analysis.
\end{itemize}

\subsection{Embedding Representations}

Three sentence-embedding representations are used:
\begin{itemize}
    \item \textbf{all-MiniLM-L6-v2}, a small distilled sentence-transformer encoder producing $384$-dimensional embeddings as a compact monolingual baseline \cite{reimers2019}.
    \item \textbf{LaBSE}, a language-agnostic BERT sentence-embedding model producing $768$-dimensional multilingual representations \cite{feng2020}.
    \item \textbf{Multilingual DistilBERT}, a compressed multilingual version of BERT \cite{sanh2019} producing $768$-dimensional representations.
\end{itemize}

Each encoder's embeddings are standardized using a \texttt{StandardScaler} fitted only on the training period, so that information from the gap or test periods does not influence the normalization parameters. Each encoder uses its own scaler, so absolute reconstruction-error magnitudes are not directly comparable across encoders; comparisons between encoders are read in terms of trends rather than absolute levels.

\subsection{Autoencoder Configuration}

For each encoder, a symmetric multi-layer autoencoder is trained on the scaled training-period embeddings. The encoder has hidden layers of sizes $512$, $384$, $256$, $128$, and $96$, with a $64$-dimensional latent bottleneck, and the decoder mirrors the encoder back to the input dimension. Each hidden layer is followed by a GELU activation. Training uses the Adam optimizer \cite{kingma2015} with learning rate $10^{-3}$, batch size $64$, and mean squared error loss, for $40$ epochs. The autoencoder optimization minimizes
\begin{equation}
\mathcal{L}_{AE} = \frac{1}{N}\sum_{i=1}^{N} \|z_i - \hat{z}_i\|^2,
\end{equation}
over the training embeddings $\{z_i\}_{i=1}^{N}$, where $\hat{z}_i$ is the reconstruction. After training, the per-sample reconstruction error is averaged over each year to obtain a yearly drift score $E_y$. The autoencoder drift threshold is set from the training-period reconstruction errors:
\begin{equation}
\tau_{AE} = \mu_{AE,\text{train}} + 0.5\,\sigma_{AE,\text{train}},\label{eq:thresh}
\end{equation}
and a year is flagged as drifted if $E_y > \tau_{AE}$. This is a sensitive threshold by design; its trade-offs are discussed in Section~\ref{sec:limitation}.

\subsection{Jensen--Shannon Divergence}

For each scaled embedding and its autoencoder reconstruction, both vectors are passed through a softmax to give pseudo-probability vectors, and the Jensen--Shannon divergence between the two is computed elementwise. Per-year averages are aggregated and a conservative threshold is set from the training-period values. As reported in Section~\ref{sec:results}, this construction produces a signal that is dominated by argmax agreement rather than by a global distributional shift; Section~\ref{sec:limitation} returns to this point.

\subsection{Kolmogorov--Smirnov and Anderson--Darling Tests}

The Kolmogorov--Smirnov test and the Anderson--Darling $k$-sample test are both applied dimension-wise between the training embedding distribution and the embedding distribution of each later year. The KS test has been widely used in unlabeled drift detection \cite{abghani2020}. Since embeddings are high-dimensional, each test produces many $p$-values, one per dimension; the Benjamini--Hochberg false-discovery-rate correction \cite{benjamini1995} is then applied. For each year, the fraction of FDR-significant dimensions is used as the drift score, and a year is flagged as drifted when this fraction exceeds $0.05$.

\subsection{Sliced Wasserstein Distance}

Direct Wasserstein distance is difficult to compute in high-dimensional spaces, so the sliced version projects the embeddings onto $K = 512$ random one-dimensional directions and averages the one-dimensional Wasserstein distance across them \cite{bonneel2015}:
\begin{equation}
\mathrm{SWD}(X_{\text{train}}, X_y) = \frac{1}{K} \sum_{k=1}^{K} W_1\!\left(X_{\text{train}}\, p_k,\ X_y\, p_k\right).\label{eq:swd}
\end{equation}

The threshold is set by splitting the training period into two random halves repeatedly and computing SWD between the halves for $B = 200$ bootstrap runs. The 95th and 99th percentiles of the resulting baseline distribution are used as drift thresholds. A year is flagged at the 95\% level if its SWD value exceeds the 95th-percentile threshold. The same procedure is also applied to the latent representations produced by the trained autoencoder, which gives a latent-space Sliced Wasserstein signal. Each baseline has $200$ bootstrap samples and produces a within-train standard deviation on the order of $10^{-3}$, which becomes important when interpreting the resulting $z$-scores (see Section~\ref{sec:discussion}).

\subsection{Tools and Reproducibility}

The experiments are implemented in Python using NumPy, Pandas, SciPy, scikit-learn \cite{pedregosa2011}, statsmodels, and PyTorch \cite{paszke2019}, with a fixed random seed of $42$. The Hugging Face \texttt{sentence-transformers} library is used to load the three encoders. All scaling parameters, autoencoder weights, and bootstrap baselines are saved per encoder so that the same yearly drift scores can be reproduced from the saved artifacts without re-encoding the corpus. The submitted project archive contains the preprocessing scripts, encoding scripts, autoencoder training code, the per-encoder drift evaluation scripts, the saved result CSV files used to produce all tables in this paper, and a README explaining the one-command reproduction entry point (see Section~\ref{sec:reproducibility}).

\subsection{Did the Project Meet Its Objectives?}
\label{sec:analysis}

This subsection and the next two relate the results back to the research questions stated in Section~\ref{sec:problem}, compare the observed behavior to what would be expected from prior work, and discuss whether the project objectives were met.

The first research question was whether semantic drift can be detected over time without labels using embedding-based representations. The reconstruction-error signal of an autoencoder trained on the training period clearly separates the test period from the training period for two of the three encoders (Table~\ref{tab:ae_period_summary}), and the yearly tables (Tables~\ref{tab:minilm_ae_yearly}--\ref{tab:labse_ae_yearly}) show a monotone increase across the test years for both MiniLM and LaBSE, with the maximum at the most recent year. The cross-encoder correlations confirm that this is a property of the data rather than an artifact of a specific encoder. The first objective is therefore met.

The second research question concerned the stability of drift signals across encoders. The AE-error signal is highly consistent across all three encoders ($r \geq 0.97$). The bootstrap-based SWD signal is less consistent (down to $r = 0.119$ for MiniLM--DistilBERT on ordinary SWD), and the dimension-wise tests are uninformative because they fire in every year and every encoder. The second objective is therefore met in the qualified sense that consistency depends strongly on which signal is used.

The third research question was which unsupervised techniques are most useful at scale. The AE-error signal is the only signal in this study that combines a graded magnitude, consistent cross-encoder behavior, and per-sample interpretability. The dimension-wise tests confirm the existence of a shift but provide no discrimination between years at this scale; SWD with a within-train bootstrap null produces uncalibrated $z$-scores; and JSD computed through softmax over standardized embeddings is dominated by argmax agreement. The third objective is therefore met.

\subsection{Comparison with Prior Work}

The dimension-wise sensitivity behavior observed here is consistent with the warnings issued in earlier surveys, which point out that two-sample tests become trivially significant at scale and that practitioners need to look at effect sizes rather than $p$-values \cite{lu2018,bayram2022}. The monotone reconstruction-error increase observed for MiniLM and LaBSE is consistent with prior reports that document embeddings can track gradual semantic shift in text streams \cite{hidalgo2019}. The non-monotone behavior of multilingual DistilBERT---in which the gap period is easier to reconstruct than the training period---is, to the author's knowledge, less directly discussed in prior work and is treated here as an empirical caution against assuming that a multilingual encoder behaves the same as a monolingual one when the underlying text is in a single language.

\subsection{Validation of Threshold and Bootstrap Choices}

Two threshold choices are validated against the resulting yearly numbers. First, the AE threshold $\tau_{AE} = \mu + 0.5\sigma$ is shown to be sensitive by design: it flags every MiniLM and LaBSE post-gap year but produces only one flag for DistilBERT. The detected pattern (gap easier than train) survives this threshold choice and is robust to setting the threshold higher (e.g.\ $\mu + 2\sigma$). Second, the SWD bootstrap null is validated by inspecting the standard deviation of the baseline: a within-train standard deviation on the order of $10^{-3}$ implies that any meaningful between-period difference will produce extremely large $z$-scores. This confirms the methodological caution that the bootstrap-based binary flag should be read as ``the year's SWD is far outside within-train variation'', not as a calibrated $p$-value.

\section{Discussion}
\label{sec:discussion}

\subsection{Embedding Representations and Autoencoder Reconstruction Error}

The three embeddings provided complementary views. MiniLM produced the clearest period-level monotone trend in autoencoder reconstruction error and was the most active flagger on the AE signal. LaBSE produced a milder but still monotone trend, with the threshold sitting close to the gap and test means. DistilBERT, the multilingual encoder, behaved differently: gap-period reconstruction was easier than training-period reconstruction, and post-gap reconstruction error was only slightly above the training level. Despite these differences in absolute behavior, the year-to-year movements of the autoencoder reconstruction error were almost the same across encoders, with $r \geq 0.97$ for every encoder pair on post-gap aggregates. This suggests that the temporal pattern picked up by the autoencoder is mainly a property of the underlying data rather than of any single embedding representation. For monitoring purposes, the choice of encoder among the three considered here is not very important, as long as the same encoder is used consistently over time.

Two properties make the autoencoder reconstruction error useful at scale. First, it is a per-sample signal that aggregates into a smooth yearly average and does not flag every year; even when the dimension-wise tests fire in every year, the AE error provides a graded magnitude. Second, it is consistent across encoders. Both properties make it a strong candidate for the central indicator in an unsupervised monitoring pipeline, especially when paired with a more conservative threshold than the one used here.

\subsection{Statistical Distance-Based Methods}

The dimension-wise tests (KS and AD) and the bootstrap-based Sliced Wasserstein Distance flagged drift in every post-gap year for every encoder. In one sense, this is consistent with the underlying language change that the AE error also captures. In another sense, the binary decisions of these tests no longer carry discriminative information at this scale.

For dimension-wise tests, the reason is straightforward. With several thousand reviews per year and a reference distribution of more than $16{,}000$ training reviews, any small but consistent change in the marginal distribution of a single embedding dimension produces a $p$-value far below any reasonable FDR threshold. Because hundreds of dimensions contribute, the fraction of FDR-significant dimensions easily crosses the drift-flag threshold of $0.05$ in every year. This is in line with the high sensitivity of dimension-wise tests in high-dimensional regimes noted in prior work \cite{lu2018,abghani2020}.

For Sliced Wasserstein Distance, the always-positive flagging is made worse by how the bootstrap baseline is built. The baseline measures the variability of SWD between two random halves of the training distribution and produces a very tight reference (standard deviation on the order of $10^{-3}$). Any meaningful difference between the training distribution and a year therefore produces $z$-scores in the range of $70$ to $250$, which is not consistent with a calibrated hypothesis-testing interpretation. The SWD magnitude is still informative as a comparable measure between years and encoders, but the bootstrap-based binary flag and the $z$-score should not be read as a $p$-value. The cross-encoder analysis also shows that ordinary SWD is much less consistent across encoders than the AE error: MiniLM and DistilBERT correlate at only $r = 0.119$ on the SWD signal even though both flag every year. The latent-space SWD inherits the same calibration issues; its value here is to confirm that the autoencoder's compression does not erase the drift signal.

\subsection{Sensitivity, Stability, and Interpretation}

At scale, the sensitivity--stability trade-off is no longer a property of the statistical method alone but also of the data setting \cite{gama2014,lu2018}. Highly sensitive methods such as dimension-wise KS, Anderson--Darling, and SWD become uniformly active under large sample sizes and lose their stability as discriminators. More conservative methods such as the AE error with a wider threshold may flag fewer years but provide a graded magnitude that stays useful as the sample size grows. The most natural way to obtain a balanced view in this setting is not to add more tests, but to combine the AE error with one carefully calibrated distributional test, such as a permutation-based SWD with a permutation null in place of a within-train bootstrap null.

The results provide evidence of distributional change in the embedding space, which is interpreted as semantic drift, but the signal should be read with care. Since the project is unsupervised, the drift signal does not directly prove that a downstream model would fail; prior work distinguishes between detecting distributional change and confirming downstream model degradation \cite{lu2018,bayram2022}. Whether a detected change is harmful depends on the downstream task. The DistilBERT result in particular shows that even the direction of the reconstruction-error signal is not universal across encoders: a multilingual encoder applied to monolingual English text may produce embeddings whose reconstruction structure differs from that of monolingual encoders. The proposed framework should therefore be seen as an early-warning system that can identify periods where the input distribution has changed and where deeper investigation is needed.

\subsection{Limitations}
\label{sec:limitation}

The proposed framework is fully unsupervised, so it cannot directly confirm whether the detected drift leads to downstream model degradation. The dataset does not contain ground-truth drift labels, which is a common difficulty in real-world concept drift evaluation where exact drift start and end times are often unknown \cite{lu2018}. Since labels are unavailable, the conditional relationship $P_t(Y \mid X)$ between inputs and labels cannot be evaluated, and the connection between detected semantic drift and actual prediction degradation cannot be checked here \cite{bayram2022}. The experiments are based on a single time-stamped review dataset, so the exact behavior of each method may vary in other domains such as news, social media, or medical text.

The drift detection results also depend on the choice of embedding model. In this project, the three encoders produced broadly similar autoencoder trends, with cross-encoder Pearson correlations of $r \geq 0.97$ on post-gap yearly aggregates. However, the absolute reconstruction-error level and, in the case of DistilBERT, even the direction of the period-to-period change differ between encoders, so the cross-encoder consistency result should be read as a within-this-family result rather than as a universal claim.

Two of the three embeddings have dimensionality $768$ and the third has $384$. High-dimensional embeddings make statistical comparison difficult, and dimension-wise tests such as KS and Anderson--Darling become overly sensitive when applied across many correlated dimensions at large sample sizes. Even with FDR correction, both tests flagged drift in every post-gap year for every encoder. Future work could either reduce dimensionality before testing or replace dimension-wise tests with a single multivariate test such as MMD \cite{gretton2012} or a kernel two-sample test. The SWD bootstrap baseline used here, with $B = 200$ repetitions, measures within-train homogeneity rather than a calibrated null for the year-versus-train comparison, which is reflected in the very small baseline standard deviation (on the order of $10^{-3}$) and the resulting yearly $z$-scores of $70$--$250$. A more appropriate null would be a permutation test that pools training and yearly samples and computes SWD on randomly relabeled splits.

The Jensen--Shannon divergence used in this pipeline is computed between the softmax of a standardized embedding and the softmax of its autoencoder reconstruction. Because the components of a standardized vector are approximately $\mathcal{N}(0,1)$, the softmax concentrates almost all probability mass on the largest few entries, and the resulting JSD is dominated by argmax agreement rather than by a global distributional shift. A more useful divergence-based signal would be computed between per-dimension histograms of embedding values, or replaced by MMD on the embeddings directly. The autoencoder threshold $\tau_{AE} = \mu + 0.5\sigma$ is also sensitive by design: a more conventional choice would be $\mu + 2\sigma$ or a high training-error quantile, which would behave more like a calibrated anomaly indicator at the cost of some sensitivity.

Finally, the experiments use yearly windows. Yearly aggregation ensures that each window contains many thousands of reviews, but it may hide shorter-term changes; a smaller window may react faster but produce noisier estimates. The methods used here also show that a distributional shift occurred without identifying the words, topics, genres, or linguistic patterns responsible; additional qualitative analysis would be needed to explain the underlying causes of detected drift.

\subsection{Code and Data Availability}
\label{sec:reproducibility}

The project archive submitted alongside this paper contains all code, configuration, and intermediate result files needed to reproduce the tables in this work. The archive follows the structure recommended in the capstone guidelines (separate \texttt{paper/}, \texttt{code/}, and \texttt{data/} directories with a top-level \texttt{README.md}) and lists a Python environment specification through a \texttt{requirements.txt} file. The main results in this paper can be reproduced from the saved per-encoder result CSV files through a single entry-point script that regenerates the tables. The full encoding and autoencoder-training stage can also be re-run on the raw IMDB corpus, which is available from its original public source. The IMDB dataset used here is publicly available; the project README documents the exact source URL, the preprocessing steps (date parsing, removal of records with invalid dates, the chronological split into training, gap, and test periods), and the variable descriptions for the saved result files.

\section{Conclusion and Future Work}
\label{sec:conclusion}

This project investigated the problem of detecting semantic drift in unlabeled, time-stamped text data. The framework represented text documents as dense embeddings, organized them into chronological time windows, and compared the semantic distributions of later periods against a historical reference period. The experiments were run on $308{,}313$ time-stamped IMDB reviews using three sentence-embedding models \cite{sanh2019,feng2020,reimers2019}, with drift signals from autoencoder reconstruction error, Jensen--Shannon divergence, dimension-wise Kolmogorov--Smirnov and Anderson--Darling tests, and Sliced Wasserstein Distance \cite{bonneel2015} applied both to the original embeddings and to the autoencoder latent space.

The strongest result is that the autoencoder reconstruction-error signal is consistent across the three encoders, with $r \geq 0.97$ on post-gap aggregates. The choice of encoder among the three considered here has little effect on the temporal pattern detected by the autoencoder, which makes the AE error a strong default indicator for unsupervised monitoring. At the same time, the dimension-wise tests, the bootstrap-based Sliced Wasserstein test, and its latent-space variant all flagged drift in every post-gap year for every encoder. At this corpus scale these tests reliably detect that a drift exists but lose their value as discriminators between years; the bootstrap-based SWD threshold also turned out not to be a calibrated null. The JSD signal, computed between standardized embeddings and their reconstructions through a softmax transformation, was effectively inactive throughout the experiments. The multilingual DistilBERT encoder additionally showed a non-monotonic reconstruction pattern in which the gap period was easier to reconstruct than the training period. The proposed framework should therefore be understood as an early-warning system rather than a direct proof of downstream model failure \cite{gama2014,lu2018,abghani2020,demello2019}, useful for motivating further investigation, qualitative analysis, targeted labeling, or model retraining when drift is detected.

Future work could extend this project in several directions. First, the detected drift signals could be compared with downstream model performance if labels become available, as a way to relate distributional drift to actual model degradation \cite{bayram2022}. Second, the bootstrap-based SWD null could be replaced by a permutation null that pools training and yearly samples, which would give interpretable $p$-values rather than the always-large $z$-scores observed here. Third, the dimension-wise tests could be replaced by a multivariate alternative such as a kernel two-sample test. Fourth, the autoencoder threshold could be replaced by a high training-error quantile. Finally, interpretability methods could identify which words, topics, or document groups contribute most to detected drift, and the framework could be extended toward adaptive systems that automatically update models when meaningful semantic change is observed.
